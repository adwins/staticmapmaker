You can add a static map to any website with simple HTML and no JavaScript. Using URL parameters you can describe the type of map that you want and Google or Mapbox will send it back.

The Static Map Maker helps you configure the parameters and gives you the code in real time. Create a Google static map or a Mapbox static map.
